        How to create a token
        
    
          
  1. Go to github.com/settings/tokens
    2. 
          
  2. Click "Generate new token (classic)"
    3. 
          
  3. Give it any name (e.g. "Caption app updater")
    4. 
          
  4. Set expiration as you like (longer = fewer renewals)
    5. 
          
  5. Under scopes, check only repo (full control of private repos)
    6. 
          
  6. Click "Generate token", copy it, paste it above, and click Save
    7. 
        

        
The token is stored only on this machine. If you ever want to revoke it, do so on GitHub and clear it here.
